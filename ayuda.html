Ayuda

  • Menu

      El menu contiene dos posibles opciones "Entrar" y "Registrate".
      Mientras el usuario no este logueado este sera el menu disponible, solo cambiara cuando el usuario ingrese a la pagina.

  • ¿Como registrarse?

      Para registrarte debes ingresar los datos a continuación. Nombre, email, contraseña y tipo de persona, pueden ser dos, moral o física.
      Moral: Si eres una empresa
      Física: Si eres una persona.

  • ¿Como loguearse?

      Para ingresar al sitio debes proporcionar tu nombre de usuario y contraseña.

  • Menu de usuario

      Una vez dentro de la pagina, tu menu cambiara al siguiente.

  • ¿Como publicar una casa?

      Para publicar una casa, dentro del menu de usuario debes de ir a "Publicar" una vez dentro tendras que ingresar todos los datos requeridos, en tipo de servicio puedes decir que vas a hacer, si esta en venta, renta, es una pension o a tratar.

  • ¿Como comprar una casa?

      Si una casa te intereso, podras utilizar el boton contactar, este le enviara una notificacion al dueño para despues realizar su venta
